What is Mandarin?

Mandarin is a standalone, manifest-driven installer for Ruby tools. Point it at any *.install.json produced by doc_generator.rb and it handles binary placement, man page installation, dependency verification, environment checks, config scaffolding, and clean uninstall â€” with full dry-run support throughout.

Actions

ActionDescription
--installCheck deps, install binary, man page, config dirs/files
--uninstallRemove binary and man pages (config never auto-deleted)
--checkVerify dependencies and show install plan â€” no file ops
--listPrint a full human-readable summary of the manifest
--dry-run is a modifier, not an action. Combine it with --install or --uninstall to simulate the full operation without touching the filesystem. Dependencies are always checked live even in dry-run mode.

Options

FlagDescription
-m, --manifest FILE*.install.json to operate on (auto-discovered if lone file in cwd)
-n, --dry-runSimulate --install or --uninstall without changing anything
-p, --prefix DIROverride install prefix (default: from manifest binary.destination)
--sudoPrefix all shell commands with sudo
--forceOverwrite existing files
-v, --verbosePrint each command as it executes
-V, --versionPrint version and exit
-h, --helpShow help and exit

Manifest Segments

The *.install.json manifest maps directly to citrus segments. Each section is independent â€” absent sections are silently skipped.

binary
source filename, installed target name, destination dir, chmod
man_pages
source file, section number, destination man dir
dependencies
ruby_version, gems[], system_bins[] â€” each optional or required
environment
required/optional env vars with hints for ~/.bashrc export lines
config_dirs
directories to create with mode bits
config_files
files to copy, with skip_if_exists safety flag
post_install_notes
human-readable manual steps shown after install completes
uninstall_notes
manual cleanup steps shown after uninstall completes

Usage Examples

# Auto-discover lone manifest in cwd, check deps
mandarin --check

# Full dry-run: see exactly what install would do
mandarin -m tool.install.json --install --dry-run

# Install to home prefix (no sudo needed)
mandarin -m tool.install.json --install --prefix ~/.local

# Install to system prefix
mandarin -m tool.install.json --install --sudo

# Overwrite an existing install
mandarin -m tool.install.json --install --force --sudo

# Dry-run uninstall â€” see what would be removed
mandarin -m tool.install.json --uninstall --dry-run

# Uninstall for real
mandarin -m tool.install.json --uninstall --sudo

# List manifest contents
mandarin -m tool.install.json --list
